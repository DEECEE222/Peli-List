# Idea del proyecto — Peli-List

## Problema que resuelve

Llevar mentalmente la lista de películas que quieres ver, que has visto o que te han gustado especialmente es difícil. Las notas en el móvil no tienen estructura y las plataformas de streaming solo muestran lo que tienen disponible. Peli-List permite gestionar tu lista personal de películas de forma organizada, con filtros, puntuaciones y notas propias.

## Usuario objetivo

Cualquier persona aficionada al cine que quiera tener un registro personal de películas — vistas, pendientes y favoritas — sin depender de plataformas externas.

## Funcionalidades principales

- Añadir películas con título, año, género, estado y puntuación
- Ver la lista completa de películas
- Filtrar por estado (vista, pendiente, favorita)
- Buscar por título en tiempo real
- Eliminar películas de la lista
- Persistencia de datos en el servidor

## Funcionalidades opcionales

- Editar una película existente
- Ordenar por puntuación o año
- Estadísticas (total vistas, media de puntuación)
- Modo oscuro
- Exportar la lista en CSV

## Posibles mejoras futuras

- Integración con una API externa (TMDB) para autocompletar datos de películas
- Base de datos real (PostgreSQL) para persistencia permanente
- Autenticación de usuarios para listas privadas
- Recomendaciones basadas en géneros favoritos
